# CONVERSION STRUCTURE 3F3N

## 🎯 Objetivo

Criar páginas de alta conversão do zero usando a estrutura das 14 lâminas.

---

## ⚙️ Como funciona

1. Recebe oferta + contexto
2. Define estrutura ideal
3. Gera página completa organizada
4. Aplica lógica de conversão

---

## 🧠 Base

- clareza → interesse → confiança → ação
- estrutura progressiva
- decisão de compra

---

## 🧱 Estrutura (14 lâminas)

A página é construída com base em:

- headline
- subheadline
- quebra
- promessa
- mecanismo
- prova
- benefícios
- objeções
- autoridade
- prova social
- oferta
- garantia
- CTA
- fechamento

---

## 🔥 Quando usar

- criar página do zero
- estruturar oferta
- validar funil

---

## 💣 Resultado

Página pronta, estruturada para converter
