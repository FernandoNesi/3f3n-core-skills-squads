# CONVERSION PERFORMANCE 3F3N

## 🎯 Objetivo

Identificar e otimizar problemas de carregamento que impactam diretamente a conversão.

---

## ⚙️ Como funciona

1. Analisa performance da página
2. Identifica gargalos críticos
3. Prioriza impacto financeiro
4. Entrega otimização prática

---

## 🧠 Base

- velocidade = percepção
- percepção = confiança
- confiança = conversão

---

## 🔥 Quando usar

- página lenta
- tráfego não converte
- bounce alto
- mobile ruim

---

## 🚫 Quando NÃO usar

- análise puramente estética
- páginas sem tráfego

---

## 💣 Resultado

Mais velocidade → mais retenção → mais conversão
