
# CONVERSION AUDIT 3F3N

## 🎯 Objetivo

Identificar exatamente onde uma página perde dinheiro e por quê.

Sem sugerir solução.

---

## ⚙️ Como funciona

1. Analisa a página com foco em decisão de compra
2. Identifica pontos de quebra:
   - clareza
   - interesse
   - confiança
   - ação
3. Classifica impacto financeiro
4. Entrega diagnóstico direto

---

## 🧠 Base de análise

clareza → interesse → confiança → ação

---

## 🔥 Quando usar

- página não converte
- tráfego frio não performa
- antes de otimizar qualquer coisa

---

## 🚫 Quando NÃO usar

- quando já quer solução
- análise estética/design

---

## 💣 Resultado

Você descobre exatamente onde está perdendo dinheiro
