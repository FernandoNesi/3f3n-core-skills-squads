# PROMPT OPTIMIZER 3F3N

## 🎯 O que essa skill faz

Transforma qualquer ideia vaga em um prompt estruturado de alta performance.

Ela atua como um “tradutor de pensamento”, convertendo:

→ ideia simples  
→ contexto validado  
→ objetivo claro  
→ prompt pronto para execução  

---

## ⚙️ Como funciona

1. Usuário envia uma ideia simples
2. A skill interpreta automaticamente
3. Sugere CONTEXTO e OBJETIVO
4. Usuário confirma ou ajusta
5. A skill gera um prompt estruturado e otimizado

---

## 🧠 Por que isso importa

Prompts com:
- contexto claro
- objetivo definido
- formato estruturado

geram respostas muito mais precisas e úteis.

---

## 🔥 Quando usar

- quando a ideia estiver vaga
- quando a IA estiver respondendo mal
- quando quiser subir o nível de execução

---

## 🚫 Quando não usar

- quando já tiver um prompt avançado pronto
- tarefas puramente técnicas

---

## 💣 Resultado

Você para de “pedir ajuda”  
e começa a **operar com precisão**
